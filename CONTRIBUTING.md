# Contributing to athene-genealogy

Thank you for your interest in this project.

## Project status

This is a personal tool developed for my own genealogical research workflow. It is published openly in the hope that others working with Obsidian and genealogical data might find it useful. Active development follows my own research needs.

## Bug reports and feature requests

Bug reports and feature suggestions are welcome via [GitHub Issues](https://github.com/rolandpd/athene-genealogy/issues). Please:

- Check existing issues before opening a new one
- For bugs: describe what you expected, what happened, and include your Obsidian and plugin version
- For feature requests: describe the use case, not just the solution

## Pull requests

I am not actively reviewing pull requests at this time. If you have a fix or improvement, feel free to open an issue first to discuss it — that avoids wasted effort on both sides.

If you fork the project and build something useful, I would be happy to hear about it.

## Code of conduct

Be respectful. This is a small project maintained in spare time.
